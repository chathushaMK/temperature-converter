# Temperature Converter

A simple browser-based temperature conversion tool for Celsius Fahrenheit. Enter a value, choose a direction, and get the converted result instantly.

## Features

- Convert Celsius to Fahrenheit
- Convert Fahrenheit to Celsius
- Clean, centered UI with responsive layout

## How to use

1. Open `index.html` in your browser.
2. Enter a number in the input field.
3. Select the conversion direction.
4. Click **Submit** to see the result.

## Project structure

- `index.html` - Markup and form layout
- `style.css` - Styling
- `index.js` - Conversion logic

## Notes

- Results are rounded to one decimal place.
- If no unit is selected, the app shows a prompt to select a unit.
